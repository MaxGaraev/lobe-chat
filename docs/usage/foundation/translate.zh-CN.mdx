# 翻译会话记录

## 翻译对话中的内容

GDZ.WORKS 支持用户一键将对话内容翻译成指定语言。选择目标语言后，GDZ.WORKS 将调用预先设置的 AI 模型进行翻译，并将翻译结果实时显示在聊天窗口中。

## 翻译模型设置

你可以在设置中指定您希望使用的模型作为翻译助手。

- 打开`设置`面板
- 在`系统助手`中找到`翻译设置`选项
- 为你的`翻译助手`指定一个模型
